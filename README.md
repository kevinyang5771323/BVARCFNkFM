# 项目简介

本项目是一款基于Spring Boot和Vue的新闻管理系统，旨在提供一个高效、便捷的新闻发布和管理平台。

## 核心功能

- 用户端：
  - 首页：展示最新新闻和热门新闻。
  - 新闻排行榜：展示热门新闻和用户收藏的新闻。
  - 收藏：用户可以收藏喜欢的新闻。
  - 个人中心：用户可以查看个人信息和操作。
  - 浏览新闻：用户可以浏览和阅读新闻内容。

- 管理端：
  - 个人中心：管理员可以查看个人信息和操作。
  - 用户管理：管理员可以对用户进行增删改查操作。
  - 排行榜管理：管理员可以管理新闻排行榜。
  - 收藏管理：管理员可以管理用户的收藏新闻。
  - 系统管理：
    - 轮播图：管理员可以管理轮播图内容。

## 适用场景

- 新闻网站
- 企业内部新闻发布平台
- 教育机构新闻管理系统

## 技术特点

- 采用Spring Boot框架，提高开发效率和系统稳定性。
- 使用Vue.js进行前端开发，提供良好的用户体验。
- 支持多用户管理，满足不同角色的需求。
- 支持新闻排行榜和收藏功能，提高用户互动性。
- 支持轮播图功能，提升页面视觉效果。
